# JK Multimedia Boilerplate

A modern, responsive HTML/CSS/Bootstrap boilerplate for web demos and projects.  
Includes a hero section, product grid, services section, contact form, back-to-top button, and fully responsive layout.

---

## Features

- Fully responsive using **Bootstrap 5**.
- Hero section with **call-to-action buttons**.
- Product/service grid layout with images and captions.
- Contact form with smooth input focus effects.
- Floating **back-to-top button**.
- Dark and light themed sections for easy customization.
- Prebuilt navigation bar and footer.
- Clean, modern design suitable for multimedia, portfolio, and small business websites.
